# 🚀 首次上传到 GitHub（傻瓜级教程）

> 只需要做一次！以后都可以双击 `push.bat` 自动推送

## 第 1 步：下载打包文件

我已经帮你把整个项目打包成 zip 了：
- 文件位置：`C:\Users\ethaneva\Desktop\术语校对工具.zip`
- 文件大小：约 14MB

## 第 2 步：解压到桌面

1. 找到桌面上的 `术语校对工具.zip`
2. **右键 → 解压到当前文件夹**
3. 确认桌面上有 `术语校对工具` 文件夹

## 第 3 步：安装 Git（如果还没装）

1. 打开浏览器，访问：https://git-scm.com/download/win
2. 点击下载按钮（会自动下载）
3. 双击安装包，**一路点 Next**（所有选项都保持默认）
4. 安装完成后点 Finish

## 第 4 步：打开命令行

1. 打开 `术语校对工具` 文件夹
2. 在文件夹空白处，**按住 Shift 键 + 右键**
3. 选择 **"在此处打开 PowerShell 窗口"** 或 **"在终端中打开"**

## 第 5 步：执行这些命令（复制粘贴就行）

在打开的黑色窗口里，**逐行复制粘贴以下命令**：

```bash
git init
```
↑ 按回车

```bash
git config user.name "wbpxy274299-ai"
```
↑ 按回车

```bash
git config user.email "wbpxy274299-ai@users.noreply.github.com"
```
↑ 按回车

```bash
git add .
```
↑ 按回车

```bash
git commit -m "初始版本：模块化术语校对工具"
```
↑ 按回车

```bash
git branch -M main
```
↑ 按回车

```bash
git remote add origin https://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@github.com/wbpxy274299-ai/terminology-tool.git
```
↑ 按回车

```bash
git push -u origin main
```
↑ 按回车

## 第 6 步：看到成功提示

如果看到类似这样的输出，就说明成功了：
```
To https://github.com/wbpxy274299-ai/terminology-tool.git
 * [new branch]      main -> main
Branch 'main' set up to track remote branch 'main' from 'origin'.
```

## 第 7 步：验证

打开浏览器，访问：https://github.com/wbpxy274299-ai/terminology-tool

如果能看到你上传的文件，就成功了！🎉

## 常见问题

### Q: 提示"git 不是内部命令"
A: Git 没装好，重新安装一下

### Q: 提示"authentication failed"
A: token 可能过期了，重新生成一个

### Q: 提示"failed to push some refs"
A: 执行这个命令强制推送：
```bash
git push -u origin main --force
```
