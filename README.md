# 游戏术语校对工具

> 多语言游戏术语校对工具，支持7种语言（日/英/韩/繁中/越南/印尼/泰），模块化架构，数据与代码分离。

## 快速开始

### Windows
双击 `start.bat`

### Mac / Linux
```bash
./start.sh
```

浏览器会自动打开 `http://localhost:8080`

## 功能说明

### 🔍 搜索
输入关键词，快速查找术语及其所有语言翻译。

### 📝 校对
粘贴文案，自动识别其中的游戏术语，并显示标准翻译对照表。

### 📦 批量校对
上传 .txt / .json / .csv 文件，逐行检查术语使用情况。

### 🔄 更新术语表
支持两种模式：
- **批量导入**：同时拖入多个语言的 Excel 文件，自动识别并合并
- **单语言更新**：选择语言，上传该语言的 Excel 文件更新术语表

更新后可下载新的 `terms.json` 文件，替换 `data/terms.json` 即可生效。

## 为什么需要启动脚本？

数据存储在 `data/terms.json`（13MB），浏览器从本地文件系统（`file://`）无法直接加载 JSON 文件（安全限制）。启动脚本会创建一个简单的本地服务器，让工具可以正常加载数据。

如果你只是想用旧版的单文件 HTML，原始文件仍保留在上一级目录：`术语校对工具_多语言聚合版.html`

## 更新术语表后的操作

1. 在工具中点击"🔄 更新术语表"
2. 上传新的 Excel 文件
3. 点击"💾 下载更新后的 JSON"
4. 用下载的 `terms_YYYY-MM-DD.json` 替换 `data/terms.json`

## 技术栈

- 纯前端（HTML + CSS + JavaScript）
- 无后端依赖
- 数据格式：JSON（从原始 HTML 内嵌数组分离）
- Excel 解析：SheetJS (xlsx)

## 注意事项

- 首次加载可能需要 1-2 秒（读取 13MB 数据）
- 搜索和校对功能完全离线，不需要网络
- 更新术语表功能需要上传 Excel 文件，也完全离线
- 所有数据存储在本地，不会上传到任何服务器
