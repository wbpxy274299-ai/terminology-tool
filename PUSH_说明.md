# 自动推送脚本

> 双击 `push.bat` 即可自动推送代码到 GitHub

## 使用前准备

### 1. 安装 Git
如果还没安装 Git，请先下载安装：
- 下载地址：https://git-scm.com/download/win
- 安装时**一路点 Next** 即可（默认选项就行）

### 2. 使用方法
**超级简单：双击 `push.bat`**

脚本会自动：
1. ✅ 检查 Git 是否安装
2. ✅ 初始化 Git 仓库（首次使用）
3. ✅ 添加所有更改的文件
4. ✅ 提交（自动生成时间戳作为提交信息）
5. ✅ 推送到 GitHub

### 3. 看到什么算成功？
```
============================================
  推送成功！
  如果配置了 Vercel，网站将自动更新
============================================
```

## 常见问题

### Q: 提示"未找到 Git"
A: 安装 Git 后重新双击 `push.bat`

### Q: 提示"没有发现更改"
A: 说明你没有修改任何文件，不需要推送

### Q: 提示"推送失败"
A: 检查网络连接，或者手动执行以下命令：
```bash
git push -u origin main --force
```

### Q: 推送后 Vercel 没更新？
A: 检查 Vercel 是否正确绑定了这个仓库

## 工作原理（好奇的话可以看看）

脚本做了这些事情：
```bash
git add .                    # 添加所有文件
git commit -m "更新 时间"     # 提交
git push -u origin main      # 推送到 GitHub
```

就这三步，脚本帮你自动完成了。
